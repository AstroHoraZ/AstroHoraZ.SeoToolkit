.NET 8 • SEO • Multilingual

AstroHoraZ SEO Toolkit

A production-ready multilingual XML sitemap generator built with ASP.NET Core (.NET 8). Designed for performance-focused, SEO-optimized modern web applications.

View Source Code on GitHub

🚀 Features

  • Dynamic XML sitemap generation
  • Multilingual support (e.g., /en, /hi)
  • Sitemap index support
  • SEO-compliant structure
  • Configurable Base URL
  • Lightweight & performance-first architecture

📌 Available Endpoints

  • /sitemap.xml
  • /sitemap-en.xml
  • /sitemap-hi.xml
  • /sitemap-index.xml

⚙ Configuration

Set your Base URL inside appsettings.json: 

"AppSettings": {
  "BaseUrl": "https://www.yourdomain.com"
}
