# AstroHoraZ Dynamic Sitemap Generator (.NET 8)

Production-ready multilingual XML sitemap generator built with ASP.NET Core.

## Features
- Dynamic sitemap.xml generation
- Multilingual support
- Sitemap index support
- SEO compliant XML structure
- Built for performance-first architecture

## Endpoints
- /sitemap.xml
- /sitemap-en.xml
- /sitemap-hi.xml
- /sitemap-index.xml
